kalibrasi pada sensor pH bertujuan untuk memeriksa ketepatan
pada sensor yang akan digunakan.

kalibrasi ini dilakukan pengaturan pada module
dengan mengatur potensio yg ada

*note : sudah dilakukan pengujian dan didapati 
        - nilai tegangan untuk pH 7 yaitu : 2,53v dengan +- 0,01v
        - nilai analog untuk pH 7 yaitu 519 dengan +- 1 

        untuk langkah pengujian berikutnya adalah dengan menggunakan prob
        dan pH buffer untuk mendapatkan nilai pH 4 yang akan digunakan untuk
        kalkulasi nilai pH_Step dan nilai pH hasil pembacaan sensor
